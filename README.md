# 视频音频提取工具

> Video Audio Extractor — 基于浏览器的视频音频提取与分段处理工具

## 简介

本工具是一款纯前端、无需后端的视频音频提取与分段处理工具。支持批量上传视频文件，自动检测音频段落（基于静音检测），并可手动编辑、切分、合并音频段，最终导出为多种音频格式。

无需安装任何软件，打开 `index.html` 即可使用。

## 功能特性

### 核心功能
- **视频音频提取**：从 MP4/AVI/MOV/MKV 等格式视频中提取音频
- **批量处理**：同时加载多个视频文件，逐个处理
- **自动检测音频段**：基于静音阈值自动检测有效音频段落

### 音频段编辑
- **手动添加片段**：在任意位置添加音频段
- **拖拽调整边界**：直接拖拽左右手柄调整起止时间
- **精准切分**：在播放指针位置一键切分音频段
- **合并片段**：将相邻音频段合并为一段
- **重命名**：为每个音频段自定义名称
- **批量重命名**：导入 TXT 文件批量重命名所有段落

### 导出功能
- 支持导出格式：**WAV / MP3 / AAC / OGG**
- 导出全部音频段或单独导出指定段

### 可视化
- **全局波形预览**：查看整个视频的音频波形，已检测段落高亮显示
- **逐段波形**：每个音频段独立显示波形
- **音频可视化特效**：
  - 频谱柱状图
  - 波形图
  - 环形频谱
  - 粒子效果

### 其他
- **撤销/重做**：支持 Ctrl+Z / Ctrl+Shift+Z 快捷键
- **键盘快捷键**：空格播放/暂停、Delete 删除当前段
- **播放指针拖拽**：直接拖拽指针快速定位

## 使用方法

### 1. 打开工具
双击 `index.html`，用浏览器（推荐 Chrome/Edge）打开。

### 2. 上传视频
- 点击上传区域，或直接将视频文件拖拽到页面中
- 支持同时选择多个视频文件

### 3. 自动检测音频段
- 调整检测参数：
  - **静音阈值**：建议 -40dB（数值越大越敏感）
  - **最小片段时长**：低于此时长的片段会被忽略
  - **静音检测时长**：连续静音超过此时长才判定为静音
- 点击「🔍 自动检测音频段」

### 4. 手动编辑
- **调整时间**：直接修改每段的时间输入框
- **拖拽手柄**：在每段波形上拖拽左右红杆调整起止时间
- **切分**：将播放指针拖到想要的位置，点击「✂️ 切分」
- **合并**：选中一段，点击「🔗 合并前段」或「🔗 合并后段」
- **重命名**：点击段名输入框直接编辑

### 5. 批量重命名（可选）
创建一个 TXT 文件，每行一个名称，然后点击「📝 导入重命名文件」。

### 6. 导出
- 选择导出格式（WAV / MP3 / AAC / OGG）
- 点击「💾 导出」导出单个段落，或「📥 导出全部音频」批量导出

## 技术说明

- 纯前端实现，无需服务器
- 音频解码基于 Web Audio API
- 波形绘制基于 Canvas API
- 可视化基于 AnalyserNode
- 兼容现代 Chrome/Edge/Firefox 浏览器

## 注意事项

- 大视频文件的音频提取需要一定处理时间，请耐心等待
- 导出功能依赖浏览器支持，建议用 Chrome/Edge 获得最佳体验
- 所有处理均在本地完成，视频文件不会上传到任何服务器
